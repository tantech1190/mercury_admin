## ğŸš¨ **Troubleshooting**

### **Problem: I don't see ANY auditor scoring**

- [ ] Do you have audits uploaded?
- [ ] Do those audits have scores? (Check audit details)
- [ ] Are audits assigned to auditors?
- [ ] Are audits marked as 'completed'?

### **Problem: I see "Score Analytics" but no auditor tables**

**Cause:** No auditors have completed audits with scores

**Solution:**
1. Go to Auditors tab
2. Create at least one auditor
3. Go to Audits tab
4. Assign audits to that auditor
5. Make sure audits have scores
6. Mark audits as completed
7. Return to Reports tab

### **Problem: Scores are all 0 or undefined**

**Cause:** Audits uploaded without scores and auto-calculation didn't work

**Solution:**
1. Check your Excel has response columns
2. Re-upload the file
3. System will auto-calculate scores
4. OR add a "Score" column manually
